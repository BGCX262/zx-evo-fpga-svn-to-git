4.6 The PIC16 port

The PIC16 port adds support for Microchip PIC MCUs with 16 bit wide instructions. Currently this family of microcontrollers contains the PIC18Fxxx and PIC18Fxxxx; devices supported by the port include:

18F: 242, 248, 252, 258, 442, 448, 452, 458

18F: 1220, 1320

18F: 2220, 2221, 2320, 2321, 2331, 23k20

18F: 2410, 2420, 2423, 2431, 2450, 2455, 2480, 24j10, 24k20

18F: 2510, 2515, 2520, 2523, 2525, 2550, 2580, 2585, 25j10, 25k20

18F: 2610, 2620, 2680, 2682, 2685

18F: 4220, 4221, 4320, 4321, 4331, 43k20

18F: 4410, 4420, 4423, 4431, 4450, 4455, 4480, 44j10, 44k20

18F: 4510, 4515, 4520, 4523, 4525, 4550, 4580, 4585, 45j10, 45k20

18F: 4610, 4620, 4680, 4682, 4685, 46k20

18F: 6520, 6585, 65j50, 6620, 6680, 66j50, 66j55, 66j60, 66j65, 6720, 67j50, 67j60

18F: 8520, 8585, 85j50, 8620, 8680, 86j50, 86j55, 86j60, 86j65, 8720, 87j50, 87j60

18F: 96j60, 96j65, 97j60

An up-to-date list of supported devices is also available via 'sdcc -mpic16 -plist'.
