8.1.3 Copy-Propagation

int f() {  
  int i, j;  
  i = 10;  
  j = i;  
  return j;  
}
will be changed to

int f() {  
  int i, j;  
  i = 10;  
  j = 10;  
  return 10;  
}
Note: the dead stores created by this copy propagation will be eliminated by dead-code elimination.
